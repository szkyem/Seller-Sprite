# Seller Sprite 商品選定ダッシュボード

GitHub Pages公開用の軽量版ダッシュボードです。

## 使い方

1. `index.html` を開きます。
2. 画面上部の「データ取り込み」で `master_merged_import.csv` を選択します。
3. 「取り込み」を押すと、ブラウザ内で親ASIN単位/ASIN単位のデータが作成されます。
4. 条件、採点設定、商品保管、CSV出力を利用できます。

## 注意

- 商品データはHTMLに埋め込まれていません。
- 読み込んだCSVはブラウザ内で処理され、サーバーには送信されません。
- 外部ライブラリは使っていないため、オフラインでもCSV取り込みできます。
- 保存商品はブラウザのlocalStorageに保存されます。別PCや別ブラウザには自動共有されません。
